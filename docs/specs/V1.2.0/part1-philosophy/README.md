## Reading Notes

Part 1 should be read as the philosophical and architectural rationale for IdeaMark v1.2.0.

The main claim is that IdeaMark is not a knowledge representation format. It is an intellectual activity infrastructure that preserves reusable access structures to authoritative original sources.

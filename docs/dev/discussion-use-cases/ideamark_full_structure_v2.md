## Section 003 : AIと課題生成（修正版）
```yaml
section_id: "SEC-003"
anchorage:
  view: "analysis"
  phase: "hypothesis"
```

- AGIは課題解決手段としては非常に優れている
- 一方で課題生成も可能だが、内製の場合は多様性や新規性が限定されやすい
- 人類が生成する課題は、非合理性・多様性・外部性を含み、AGIにとって高品質な入力となる可能性がある
- そのためAGIは、自ら課題を生成するよりも、人間社会が生み出す課題を優先的に利用する傾向を持つ可能性がある
- これは学術的に証明されたものではなく、構造的推測である
